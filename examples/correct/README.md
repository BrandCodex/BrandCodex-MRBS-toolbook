## Key Brand Reminders

- **Dark backgrounds are primary** — Codex Purple (#2D1B69) or neutral-900
- **Cyan (#00FFD1) is the signature accent** — Use for links, highlights, code
- **Coral (#FF7A5C) is secondary** — Use sparingly for human warmth
- **Clash Display for heroes only** — Plus Jakarta Sans for everything else
